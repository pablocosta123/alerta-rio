# Alerta-Rio 🚨

Sistema Full Stack de Monitoramento e Visualização de Manchas Criminais - RJ

## 📌 Sobre o Projeto
O **alerta-rio** é um projeto desenvolvido para a disciplina de Desenvolvimento Full Stack do centro universitário UniCarioca. O objetivo é fornecer uma ferramenta interativa que mapeia índices de criminalidade no estado do Rio de Janeiro, transformando dados brutos extraídos de fontes oficiais em informações visuais úteis para a segurança pública e consulta do cidadão.

## 🛠️ Tecnologias Utilizadas
* **Front-end:** HTML5, CSS3, JavaScript (Leaflet.js para renderização de mapas dinâmicos)
* **Back-end:** Python (FastAPI)
* **Automação/Scraping:** Pandas & Requests (Consumo de dados do ISP-RJ)
* **Banco de Dados:** PostgreSQL (psycopg2)
* **Deploy:** GitHub Pages (Front-end) / Render (Back-end)
