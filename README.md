# alerta-rio 🚨

 **Sistema Full Stack de Monitoramento e Visualização de Manchas Criminais - RJ** 
 ## 📌 Sobre o Projeto O **alerta-rio** é um projeto de extensão desenvolvido para a disciplina de Desenvolvimento Full Stack da UniCarioca. O objetivo é fornecer uma ferramenta interativa que mapeia índices de criminalidade no estado do Rio de Janeiro, transformando dados brutos em informações visuais úteis para a segurança do cidadão. 
 
 ## 🛠️ Tecnosssslogias Utilizadas - 
 
 **Front-end:** HTML5, CSS3, JavaScript (Leaflet.js para mapas) - 
 **Back-end:** Python (Flask/FastAPI) - 
 **Automação/Scraping:** Selenium & Pandas - 
 **Banco de Dados:** SQL Server - 
 **Deploy:** GitHub Pages (Front) / Render (Back)
